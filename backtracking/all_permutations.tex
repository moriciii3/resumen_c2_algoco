\subsubsection{Generación de Todas las Permutaciones}

Este problema consiste en generar y listar todas las posibles permutaciones de una cadena de caracteres dada, en orden lexicográfico. Utiliza el algoritmo \texttt{std::next\_permutation} dentro de un bucle \texttt{do-while} para generar todas las $N!$ permutaciones de forma eficiente.

\paragraph{Complejidad:}
* \textbf{Tiempo:} $O(N \cdot N!)$, ya que se generan $N!$ permutaciones y cada una cuesta $O(N)$.
* \textbf{Espacio:} $O(N \cdot N!)$, para almacenar todas las permutaciones.
